My Apps

A Seletion Of My Work

E-Commerece Shopping Cart

An E-Commerece application that allows the user to choose from a small list of items and add them to the dynamic shopping cart. The user may then interact the items placed in the cart and adjust their quantities. The items, once selected for purchased, are then sent to a Firebase backend server. The application was built using React Redux and styled with HTML and Module.CSS.

***Please Note: The Error message you see at the top of the app is due to Firebase changing their security credentials as of 03/22/2022, so the error persists, however, the code is sound.***

Expense Tracker

An application that allows a user to add expenses to a dynamic chart that will graph and track the quantity and timestamp of added expenses by year and amount. The visual graph displays to the user the frequency of expenses per month. The app was built with React.js and focused on the fundamentals of React.

Food Menu App

An application that allows a user to browse a menu of food items and select them to be added to the checkout cart. Once in the cart, the items' quantities can be manipulated to the user's command. The app was built using ContextAPI in React and React Hooks.
